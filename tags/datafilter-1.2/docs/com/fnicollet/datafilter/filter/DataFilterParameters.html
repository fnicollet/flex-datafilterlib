Classpublic class DataFilterParameters
InheritanceDataFilterParameters flash.events.EventDispatcher

Configuration object for the DataFilter. Can be instanciated with MXML or in AS3 using an object in the constructor. Dispatches event when his values are being changed (through binding for example). Every DataFilter listen to this events
